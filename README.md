Анализ данных Формулы 1

Статистический анализ времени пит-стопов в Формуле 1.

- Проверка нормальности распределения признаков
- Сравнение средних значений с помощью t-тестов
- Дисперсионный анализ (ANOVA)
- Проверка независимости признаков (χ²-тест)

Используемые методы
- t-тест (парный и непарный)
- Однофакторный и двухфакторный ANOVA
- Критерий хи-квадрат
- Точный тест Фишера
- Критерий Шапиро-Уилка
- Критерий Левена

Результаты

Данные распределены **ненормально** (p < 0.05)

t-тесты
- Непарный t-test: сравнение McLaren и Red Bull
- Парный t-test: сравнение времени пит-стопов Макса Ферстаппена в 2019 и 2022
- Сравнение признаков с неравными дисперсиями (McLaren и Force India)

ANOVA
- Модель 1: AvgPitStopTime ~ Constructor
- Модель 2: AvgPitStopTime ~ Constructor + Season

Хи-квадрат
- Анализ распределения сезонов
- Анализ зависимости между конструктором и сезоном

Точный тест Фишера
- Анализ связи между командой и количеством пит-стопов
